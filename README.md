
# Parcial 3 corte paradigmas de programacion.


##  Descripción de Archivos

### 1. `1_Concurrencia_CalculoPi.pdf`
**Descripción:** Documento donde se realiza el diseño de regresion lineal usando paradigmas de concurrencia y calculoPi.

### 2. `2_Aspectos.pdf`  
**Descripción:** Documento donde se realiza el diseño de regresion lineal usando paradigmas de ascpectos.

### 3. `3_Comparacion_Rust_VS_Python.pdf` 
**Descripción:** Comparacion de Rust y Pyhon frente a la implementacion de regresion lineal.

### 4. `Cargo` 
**Descripción:** Archivo de configuración de Cargo (gestor de paquetes de Rust). Define dependencias y metadatos del proyecto Rust.

### 5. `Regresion_Lineal_Rust.ps1`
**Descripción:** Script que implementa regresión lineal en **Rust**. Contiene el código principal para calcular y analizar modelos de regresión lineal.

### 6. `Regresion_Lineal_Rust` 
**Descripción:** Archivo ejecutable o código fuente de Rust para regresión lineal. Versión compilada o fuente del proyecto de regresión.

##  Cómo Ejecutar los Códigos

### Requisitos Previos

#### Para Rust:
- Instalar [Rust]
- Verificar instalación: `rustc --version` y `cargo --version`

#### Para Python:
- Instalar [Python]
- Instalar dependencias necesarias:
  ```bash
  pip install numpy matplotlib scipy
  ```
